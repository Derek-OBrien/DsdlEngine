Dsdlengine

Wrapper engine for creating games using SDL2.0

View project on GitHub

Welcome to the Dsdlengine Page.

As part of my final year at Limerick Institute of Technology studying Bachelor of Science (Honours) in Computing with (Games Design and Development), I had to complete a yearlong project that would enhance the game industry, for this I decided to build an engine framework around the already popular Simple Direct Media Layer (SDL) library. The framework is cross platfom and works with Android and Windows

Feamework Features

  • Cross platform: Windows and Android
  • Sprites
  • Layers
  • Scenes
  • Scene management system
  • Xml loading and saving
  • User interface elements (Buttons, Labels)
  • Physics with Box2D
  • Rendering system
  • Audio management
  • Input management
