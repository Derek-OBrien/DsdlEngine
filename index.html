Welcome to GitHub Pages.

As part of my final year at Limerick Institute of Technology studying Bachelor of Science (Honours) in Computing with (Games Design and Development), I had to complete a yearlong project that would enhance the game industry, for this I decided to build an engine framework around the already popular Simple Direct Media Layer (SDL) library.

FrameWork Features

  • Cross platform: Windows and Android
  • Sprites
  • Layers
  • Scene management system
  • Xml loading and saving
  • User interface elements (buttons, labels)
  • Physics with Box2D
  • Rendering system
  • Audio management
  • Input management
